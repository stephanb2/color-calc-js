h1. Macbeth Color Calculator

p. Try the calculator "here.":http://www.stephanb.co.uk/color/

h2. Description

p. This calculator displays colorimetric data for color patches of the "Macbeth ColorChecker.":http://en.wikipedia.org/wiki/ColorChecker It is inspired by Bruce Lindbloom's "color calculators.":http://www.brucelindbloom.com/index.html?ColorCalculator.html

p. Colorimetric data is calculated from spectral data in 10nm bands. CIE illuminants data has been averaged to 10nm. You can compare the results with the values on the "Wikipedia page":http://en.wikipedia.org/wiki/ColorChecker by choosing: Illuminant: C, Standard Observer: CIE 1931.

p. Plotting is done with jQuery "Flot.":http://www.flotcharts.org/ Some testing is done with "QUnit.":http://qunitjs.com/
