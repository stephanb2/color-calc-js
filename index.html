Yellow triangles indicate out-of-gamut colors.

The Macbeth ColorChecker is a device used for color calibration. This calculator lets you explore its appearance under a variety of lighting conditions.

All the calculations are done from averaged spectral reflectance data collected by Danny Pascale. The values produced by this calculator will differ slightly from official values published by X-Rite, or calculated by Bruce Lindbloom. One of reasons being that the calculation are done in 10nm bands instead of the usual 5nm.

This application should run in any modern browser that supports HTML5 Canvas (Chrome, Firefox and Internet Explorer 9+). It is using jQuery and Flot. The source code is on GitHub.

Unit testing is done with QUnit. The qunit-assert-close plugin was extended to traverse arrays. This allows to check vector and matrix calculations with a given error tolerance.

This calculator is inspired by Bruce Linbloom's color calculators.
